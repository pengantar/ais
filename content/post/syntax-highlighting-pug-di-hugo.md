Secara default, Hugo nggak mendukung syntax highlighting untuk Pug. Contohnya aja seperti di bawah ini:

```pug 
doctype html 
html 
  head 
    title Hello world
    meta(charset="utf-8")
  body 
    h1.text-center Hello world
```

Maka, kita akali dengan menggunakan syntax highlighting Javascript:

```javascript
doctype html 
html 
  head 
    title Hello world
    meta(charset="utf-8")
  body 
    h1.text-center Hello world
```
